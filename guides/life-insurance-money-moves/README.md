# Life Insurance Money Moves

Tools, guides, calculators, and templates that teach families how to build tax-free wealth and generational security using life insurance.

## What's Inside
- guides/: Educational guides and quick-start resources
- calculators/: Simple calculators and scripts
- marketing/: Social media assets, post templates, and branding
- landing_page/: Simple landing page (HTML/CSS/JS)
- leads/: Lead capture examples and intake scripts

## Usage
- Use GitHub Pages to host `landing_page/index.html` for a free landing page.
- Update files and publish from this repository.
- Share `guides/` PDFs or markdown as lead magnets.
